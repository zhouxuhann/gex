# GEX Web

Isolated Next.js + Tailwind frontend scaffold for the hosted GEX product.

## Purpose

This folder is intentionally separated from the existing desktop-oriented Python application under `src/` so both can evolve in parallel.

## Commands

```bash
npm install
npm run dev
```

## Scope

- Next.js App Router setup
- Tailwind CSS setup
- basic landing page for the hosted product track
- no coupling to the existing desktop UI
